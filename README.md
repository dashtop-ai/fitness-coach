# Fitness Coach

AI-powered fitness coach for Dashtop. Get daily workouts, track your progress, and get personalized coaching tips.

## Features

- **Daily Workouts** — 6 exercises tailored to your level with checkboxes to track completion
- **Three Levels** — Beginner, Intermediate, Advanced with different exercise programs
- **Progress Tracking** — Visual progress bar and completion celebration
- **Weekly Log** — Color-coded week view, streak counter, and recent workout history
- **Stats Dashboard** — Total workouts, minutes, calories, and streak at a glance
- **AI Chat** — Ask about form, rest times, alternatives, nutrition, and more

## Install

Install in Dashtop via Settings > Packages, or:

```bash
npm install
npm run pack
# -> dist/dashtop-ai-fitness-coach-v1.0.0.dashtop
```

## Develop

```bash
npm install
npm run dev
```
